# HexChessRL

Reinforcement Learning explorations for "Can Chess With Hexagons".

## Missing Features:

- [ ] En passant captures for pawns.
- [ ] Pawn promotion.
- [ ] Check & Checkmate checks, the game now concludes only when the king is captured.
- [ ] Restart feature.
- [ ] Powerful AI players.
